# Student Grade Tracker

A simple web application for tracking student grades and GPA.

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Backend: PHP 7.4+
- Database: MySQL 5.7+

## Setup

1. Import `db/schema.sql` into MySQL
2. Configure DB credentials in `api/db.php` if needed
3. Serve with Apache/XAMPP pointing to project root
4. Open `index.html` in browser

## Features

- User registration and login
- Add grades per course
- Automatic GPA calculation
- Course listing page
- Print grade report
